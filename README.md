## AlyceHealth Front 사전 과제
: 뉴스 검색 및 뉴스기사 즐겨찾기, 로그인 기능 개발

---
## Use Stack
- React
- Typescript
- Styled-Components
- axios
- antd-Design

---
## How to Run
```js
$ cd ~/alycefront_task
$ npm install / yarn install
$ npm start / yarn start
```
---
## Trouble Shooting

- Redux를 사용하지 않고 localStorage를 통해 로그인 정보 / 즐겨찾기 기사 데이터를 수집하고 관리합니다.
- apiKey는 gitignore에 저장하여 git에 올리지 않았습니다.
